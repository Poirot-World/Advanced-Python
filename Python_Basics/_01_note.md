# 身份运算符 is
1. 身份运算符：用于比较两个对象的内存地址是否一致——是否是对同一个对象的引用

2. `is`和`==`的区别：  
`is`用于判断两个变量引用对象是否为同一个  
   `==`用于判断引用变量的值是否相等
3. 例子
```python
>>> a = [1, 2, 3]
>>> b = [1, 2, 3]
>>> b is a 
False
>>> b == a
True
```
